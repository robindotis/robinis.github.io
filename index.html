CSS Naked Day

3 March, 2023

The 9th April is CSS Naked Day. On this day many bloggers remove all styling from their blog. The idea is to promote awarenes of web standards. A well designed site, following web standards, will work well without any styling. A follow-on aim is to raise awareness of accessibility, since the plain, unstyled website will feel more how someone with a screen reader and other assistive technology will experience the site.

In preparation for this, have updated my theme picker to incldue a hard coded "Naked" CSS theme. If desired, this theme will be forced on CSS Naked Day, with no alternative for the end user.

The hundredth idiot

22 February, 2023

The hundredth idiot is a Nariscene proverb described in the Culture series book Matter by Iain M. Banks.

One hundred idiots make idiotic plans and carry them out. All but one fail. The hundredth idiot, whose plan succeeded through pure luck, is immediately convinced he's a genius.

As told by the Nariscene space vessel "The hundredth idiot".
