Publishing Process (see publish.ps1):

DONE

  1. Locally zip contents of _site folder ("zip _site site.zip")
  2. Delete remote folder (ssh xxx@ssh.phx.nearlyfreespeech.net rm -rf xxx.com/*) . See here: https://www.siteground.com/tutorials/ssh/deleting/
  3. Copy to remote server using scp (scp -ra /path/to/local/storage user@remote.host:/path/to/copy/ => scp -ra ./site.zip user@remote.host:/xxx.com/site.zip). See here: https://stackoverflow.com/questions/17828236/how-to-copy-a-directory-from-local-machine-to-remote-machine
  4. Unzip site.zip (ssh xxx@ssh.phx.nearlyfreespeech.net unzip www.xxx.com/site.zip ) see here: https://www.siteground.com/tutorials/ssh/archives/
  5. Remove zip file
